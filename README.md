# Data Dashboard

CSV, Excel `.xlsx`, 또는 `Ctrl+V`로 붙여넣은 표 데이터를 브라우저에서 바로 분석하는 React/Vite 웹앱입니다. 접속하면 기본 샘플 CSV들이 탭으로 열리고, 여러 데이터셋은 탭으로 동시에 전환할 수 있습니다.

## 로컬 실행

```bash
npm install
npm run dev
```

실행 후 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:5173/
```

외부 기기나 다른 네트워크 인터페이스에서 접근해야 하면 다음처럼 실행할 수 있습니다.

```bash
npm run dev -- --host 0.0.0.0
```

## 기본 샘플 데이터

페이지에 접속하면 repo에 포함된 기본 CSV가 자동으로 탭에 표시됩니다.

- `Sales sample`
- `Inventory sample`
- `Customers sample`

이후에도 CSV/Excel 업로드나 Ctrl+V 붙여넣기로 탭을 추가할 수 있습니다.

## 사용법

### 파일 업로드

1. 화면 상단의 파일 선택 영역을 누릅니다.
2. `CSV` 또는 Excel `.xlsx` 파일을 선택합니다.
3. 업로드가 끝나면 파일별 탭이 생성되고, 데이터 요약, 컬럼 선택, 통계, 그래프, 미리보기가 표시됩니다.
4. 여러 CSV 파일을 한 번에 선택하면 파일마다 탭이 생성됩니다.
5. Excel 파일은 시트마다 탭이 생성됩니다.

### Ctrl+V 붙여넣기

1. Excel, Google Sheets, 웹 표 등에서 분석할 표 범위를 복사합니다.
2. 웹앱의 붙여넣기 영역을 클릭합니다.
3. `Ctrl+V`로 붙여넣습니다.
4. 첫 번째 행은 기본적으로 컬럼명으로 처리됩니다.
5. 붙여넣기를 다시 하면 `Pasted table 2`, `Pasted table 3`처럼 새 탭이 추가됩니다.

## 분석 기능

### 데이터 방향

분석 설정에서 `원본`, `전치`, `매트릭스`를 선택할 수 있습니다. `전치`는 행과 열을 바꾼 형태로 통계, 그래프, 미리보기를 다시 계산합니다. 원본 데이터는 변경하지 않으며, 선택한 데이터 방향은 탭마다 따로 저장됩니다.

예를 들어 첫 컬럼이 `Metric`이고 행에 `Sales`, `Profit`이 있으며 열에 `Q1`, `Q2`가 있는 표는 전치 후 `Metric / Sales / Profit` 컬럼과 `Q1`, `Q2` 행으로 분석할 수 있습니다.

### 매트릭스 표

첫 번째 열이 시리즈 또는 Y축 이름이고, 나머지 열 머리글이 X축 값이며, 내부 셀이 실제 값인 표는 `매트릭스`를 선택합니다.

```csv
Metric,Jan,Feb
Sales,100,120
Profit,20,30
```

`매트릭스` 모드는 내부적으로 아래처럼 `X / Series / Value` 긴 형태로 변환하고, X축별 여러 시리즈 막대/라인 그래프를 그립니다.

```csv
X,Series,Value
Jan,Sales,100
Feb,Sales,120
Jan,Profit,20
Feb,Profit,30
```


### 단일 컬럼 분석

컬럼 하나를 선택해 통계와 그래프를 확인합니다.

- 숫자 컬럼: 개수, 결측값, 합계, 평균, 중앙값, 분산, 표준편차, 최솟값, 최댓값
- 문자/범주 컬럼: 개수, 결측값, 고유값 수, 상위 빈도값
- 표시 지표 체크박스로 통계 패널에 보여줄 항목을 선택할 수 있습니다.

### 두 컬럼 관계 분석

두 개의 컬럼을 선택해 관계를 확인합니다.

- 숫자 + 숫자: 산점도 또는 x값 기준 라인 차트
- 범주 + 숫자: 그룹별 평균/합계/개수/최솟값/최댓값/분산/표준편차 기반 그래프


### 그래프 타입

선택한 데이터 형태에 따라 사용할 수 있는 그래프 타입이 달라집니다.

- 숫자 컬럼: 막대, 라인, 히스토그램
- 문자/범주 컬럼: 막대, 파이, 라인
- 숫자 + 숫자: 산점도, 라인
- 범주 + 숫자: 막대, 파이, 라인

범주 + 숫자 관계에서는 `그래프 지표` 선택으로 평균, 합계, 개수, 최솟값, 최댓값, 분산, 표준편차 중 어떤 값으로 그래프를 그릴지 정할 수 있습니다. 막대형 그래프는 `세로` 또는 `가로` 방향을 선택할 수 있습니다. 차트 타입, 지표 선택, 차트 방향은 탭마다 따로 저장됩니다.

## 탭 사용

- 탭을 클릭하면 해당 데이터셋으로 분석 화면이 전환됩니다.
- 탭의 `×` 버튼을 누르면 필요 없는 데이터셋을 닫을 수 있습니다.
- 마지막 탭을 닫으면 빈 상태로 돌아갑니다.

## GitHub Pages

`main` 브랜치에 push되면 GitHub Actions가 정적 사이트를 빌드하고 GitHub Pages에 배포합니다.

예상 배포 주소:

```text
https://nowcika.github.io/test_sql/
```

저장소 Settings → Pages에서 Source가 GitHub Actions로 설정되어 있어야 합니다. Private 저장소에서 Pages를 쓰려면 GitHub 계정/조직 플랜이 이를 지원해야 하며, 지원하지 않는 경우 저장소를 public으로 전환해야 합니다.
