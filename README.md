# Data Dashboard

CSV, Excel `.xlsx`, 또는 `Ctrl+V`로 붙여넣은 표 데이터를 브라우저에서 바로 분석하는 React/Vite 웹앱입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

실행 후 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:5173/
```

외부 기기나 다른 네트워크 인터페이스에서 접근해야 하면 다음처럼 실행할 수 있습니다.

```bash
npm run dev -- --host 0.0.0.0
```

## 사용법

### 파일 업로드

1. 화면 상단의 파일 선택 영역을 누릅니다.
2. `CSV` 또는 Excel `.xlsx` 파일을 선택합니다.
3. 업로드가 끝나면 데이터 요약, 컬럼 선택, 통계, 그래프, 미리보기가 표시됩니다.

### Ctrl+V 붙여넣기

1. Excel, Google Sheets, 웹 표 등에서 분석할 표 범위를 복사합니다.
2. 웹앱의 붙여넣기 영역을 클릭합니다.
3. `Ctrl+V`로 붙여넣습니다.
4. 첫 번째 행은 기본적으로 컬럼명으로 처리됩니다.

## 분석 기능

### 단일 컬럼 분석

컬럼 하나를 선택해 통계와 그래프를 확인합니다.

- 숫자 컬럼: 개수, 결측값, 합계, 평균, 중앙값, 최솟값, 최댓값
- 문자/범주 컬럼: 개수, 결측값, 고유값 수, 상위 빈도값

### 두 컬럼 관계 분석

두 개의 컬럼을 선택해 관계를 확인합니다.

- 숫자 + 숫자: 산점도
- 범주 + 숫자: 그룹별 평균/합계 기반 막대 그래프

## 참고

- 데이터는 브라우저 메모리에서만 처리됩니다.
- 로그인, 서버 저장, 실제 MySQL 연결은 현재 버전에 포함되어 있지 않습니다.
- Excel 파일은 첫 번째 시트를 기본 분석 대상으로 사용합니다.
